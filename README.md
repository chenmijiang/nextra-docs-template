# Nextra Docs Template with Tailwind CSS

> This project is cloned from [Nextra Docs Template](https://github.com/shuding/nextra-docs-template), with some modifications.
>
> - nextra -> v4
> - nextjs -> v15
> - react -> v19
> - tailwindcss -> v4
> - use app router
